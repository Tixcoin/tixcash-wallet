# Tixcash Wallet

Desktop and mobile wallet application for managing TXC assets.

## Overview

Tixcash Wallet provides a secure, user-friendly interface for:

- Sending and receiving TXC tokens
- Viewing transaction history
- Participating in staking and earning rewards
- Interacting with Tixcash smart contracts
- Managing multiple accounts

## Security

- Private keys are stored locally and never transmitted
- BIP-44 compliant HD wallet derivation
